🚫 Ad & Tracker Blocking

Blocks ads and trackers using AdGuard, ABPVN, and 1Hosts lists. Auto-updates every hour.

🌍 ECS – Location Optimization

Sends your network location so CDNs return the closest server for faster page loads.

🔄 DNS Redirect

Redirect domains to custom CNAME records. Route traffic to a preferred CDN endpoint.

🛡️ Geo-Bypass

Detects geo-blocked domains and automatically falls back to Mullvad DNS to bypass them.

⚡ Query Filtering

Drops unnecessary query types (AAAA, PTR, HTTPS) at the edge to reduce upstream load.

🔐 Private TLD Block

Prevents internal queries like .local, .internal, .test from leaking to the internet.
